# Telegram Bot

Простой Telegram-бот на Python.

## Установка

1. Клонируйте репозиторий и перейдите в папку проекта:

```
git clone https://github.com/1ntima/TGbot_API-Perplexity.git
cd telegram-bot
```

2. Установите зависимости:

```
pip install -r requirements.txt
```

## Настройка

1. Скопируйте файл с примером конфигурации:

```
cp config.exemple.py config.py
```

2. Откройте `config.py` и подставьте свои API токены и необходимые Telegram ID.

Файл `config.py` не загружается в репозиторий, так что ваши ключи останутся приватными.

## Запуск

```
python `Perplexity.py`
```
